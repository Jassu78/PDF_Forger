# Font for DOCX→PDF conversion

The build downloads Roboto-Regular.ttf automatically. If the download fails (e.g. offline), place `Roboto-Regular.ttf` here manually:

1. Download from: https://github.com/google/fonts/blob/main/apache/roboto/static/Roboto-Regular.ttf
2. Save as `Roboto-Regular.ttf` in this directory
